What you get

Built for real agentic coding work

Everything stays local. You bring the model. Dendrite handles the team.

Local-first, bring your own model

Works with an Anthropic key, any OpenAI-compatible key, a Grok / Codex / Copilot subscription, or a local model via Ollama. Nothing leaves your machine except the prompts you send to the model you connect.

A persistent brain

Learns how you work and remembers across sessions — memory, skills, and artifacts that stick around so agents pick up where you left off.

Parallel agents

Run a team of agents that can work in isolated git worktrees — so they don’t step on each other’s changes.

Phone companion

An installable phone companion lets you watch and steer runs from your phone — check progress and nudge agents without sitting at the desk.

Any project folder

Point Dendrite at any project folder and supervise agents working inside it — features, fixes, refactors, research — right where your code already lives.

Get started

Install in a few minutes

No admin rights. No installer wizard maze. Just download and run.

  1. 1

    Download Setup.exe

    Grab the Windows installer from the download button. It’s a single Setup.exe for early builds.

    Download for Windows
  2. 2

    If Windows SmartScreen warns

    Early builds are unsigned, so Windows may show a SmartScreen warning. Click More info, then Run anyway.

    Why? Unsigned early build — expected during development. The app installs without admin privileges.
  3. 3

    Install (no admin needed)

    Run the installer. It installs without requiring administrator rights, so you can get going on a locked-down machine.

  4. 4

    Connect a model & open a folder

    Launch Dendrite Studio, connect the model you already use (Anthropic, OpenAI-compatible, Grok / Codex / Copilot, or Ollama), open any project folder, and start supervising agents.

What you’ll need

  • Windows PC
  • A model connection: API key, subscription, or local Ollama
  • A project folder to work in
