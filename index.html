The workspace

From first prompt to finished change.

Tendrite gives your agent team a shared operating surface: projects, missions, memory, source control, and the editor all connected.

⌘

Run a team, not a chat tab

Assign work to agents, give each one a role, and let them work in isolated git worktrees while you keep the whole mission in view.

◎

Local project memory

Keep the decisions, conventions, and useful artifacts that make the next session faster.

◈

Provider-ready

Connect an API key, compatible account, subscription, or local model through Ollama.

↯

Source control included

Inspect diffs, coordinate branches, and bring changes back into your project without leaving the workspace.

Know what you get

Free is the complete local core. Pro adds the connected layer.

You can use Tendrite’s core desktop workspace without a license. Pro is for carrying context across projects, agents, and devices.
